# خيارات النشر المتاحة

## الخيارات المجانية الآمنة برابط دائم:

### 1. **GitHub Pages** (الأفضل والأسهل) ⭐
- **المميزات:**
  - مجاني تماماً
  - رابط دائم: `https://username.github.io/repo-name`
  - آمن جداً
  - مدعوم من GitHub

**الخطوات:**
1. أنشئ حساب GitHub (إن لم يكن موجود)
2. أنشئ repository جديد باسم `neonatal-jaundice`
3. اضغط على Settings → Pages
4. في "Source" اختر `main` branch و `/docs` folder
5. انسخ الملفات من `dist/` إلى `docs/`
6. اضغط push
7. الرابط: `https://your-username.github.io/neonatal-jaundice`

---

### 2. **Vercel** (سهل مثل Netlify)
- **المميزات:**
  - مجاني تماماً
  - بنفس سهولة Netlify
  - رابط دائم
  - أسرع CDN

**الخطوات:**
1. اذهب إلى https://vercel.com
2. سجل دخول باستخدام GitHub
3. اضغط "New Project"
4. اختر repo الخاص بك
5. اضغط Deploy

---

### 3. **Cloudflare Pages**
- **المميزات:**
  - مجاني تماماً
  - سريع جداً
  - آمن جداً

**الخطوات:**
1. اذهب إلى https://pages.cloudflare.com
2. سجل دخول باستخدام حساب Cloudflare
3. اختر GitHub repository
4. اضغط Deploy

---

## توصيتنا:
**استخدم GitHub Pages** - إذا كان لديك حساب GitHub
**أو Vercel** - إذا أردت بديل موثوق
